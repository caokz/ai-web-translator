# 📚 导出功能使用指南

## 概览
AI Web Translator 现在支持**两种导出模式**，让你灵活选择适合的内容提取方式。

---

## 🎯 两种导出模式

### 模式1️⃣: 原网页内容 (Raw)
用于保存完整的网页内容参考。

**特点**:
- ✨ 保留原网页结构和格式
- 📄 导出完整内容
- ⚡ 处理速度快（无需调用AI）
- 🎯 适合需要完整信息的场景

**何时使用**:
- 保存重要参考资料
- 存档网页内容
- 需要原始结构和细节

---

### 模式2️⃣: AI提炼 (Summarize)
用于快速浏览和记笔记。

**特点**:
- 🤖 智能提取关键内容
- 📝 精简总结要点
- 🧠 使用配置的AI模型
- 🎯 适合快速阅读

**何时使用**:
- 快速浏览内容要点
- 记取笔记和总结
- 提取主要信息
- 需要简洁版本

---

## 🚀 使用步骤

### 步骤1: 打开网页
在浏览器中打开你想导出的网页。

### 步骤2: 打开扩展程序
点击AI Web Translator扩展图标打开弹窗。

### 步骤3: 选择导出模式
在 **"📄 导出模式"** 部分选择：
- 🌐 **原网页内容** - 完整导出原网页
- 🤖 **AI提炼** - 使用AI总结内容

```
┌─────────────────────────────┐
│ 📄 导出模式                 │
├─────────────────────────────┤
│ ◉ 原网页内容  ○ AI提炼      │
├─────────────────────────────┤
│ ✨ 保留原网页结构...        │
└─────────────────────────────┘
```

### 步骤4: 点击导出按钮
点击 **"📥 导出为Markdown"** 按钮。

### 步骤5: 等待并保存
- 导出中显示 "导出中..."
- 完成后浏览器自动下载Markdown文件

---

## 📥 导出文件说明

### 文件名格式
```
{网页标题}-{日期}.md
示例: AI-Web-Translator-2024-12-07.md
```

### 文件内容结构

#### 原网页内容模式
```markdown
[网页内容...]

---

> **来源:** [网页标题](https://example.com)
> **导出模式:** 原网页导出
> **导出时间:** 2024-12-07T10:30:00.000Z
```

#### AI提炼模式
```markdown
[AI提炼的内容总结...]

---

> **来源:** [网页标题](https://example.com)
> **导出模式:** AI提炼
> **导出时间:** 2024-12-07T10:30:00.000Z
```

## ⌨️ 快捷键

| 快捷键 | 功能 | 模式 |
|--------|------|------|
| **Alt+E** | 快速导出 | 原网页内容(Raw) |
| **Alt+T** | 切换翻译 | N/A |

**提示**: 使用快捷键时，默认使用 **原网页内容** 模式。

---

## ⚙️ 配置要求

### 原网页内容模式
✅ 无需特殊配置
✅ 立即可用

### AI提炼模式
需要配置API Key：

1. 点击弹窗底部的 **"设置"** 按钮
2. 选择你偏好的AI模型
3. 输入对应的API Key
4. 保存设置

**支持的模型**:
- 🔍 DeepSeek
- 🌙 Kimi
- 🤖 OpenAI
- 🔮 通义千问
- 💡 文心一言

## 💡 使用技巧

### 技巧1: 选择合适的模式
| 场景 | 推荐模式 |
|------|---------|
| 保存参考资料 | 原网页内容 |
| 记笔记 | AI提炼 |
| 长文章 | AI提炼（更简洁） |
| 重要数据 | 原网页内容 |
| 快速浏览 | AI提炼 |

### 技巧2: 导出后处理
导出的Markdown文件可以：
- 📝 在Markdown编辑器中编辑
- 📚 导入到笔记应用（Notion、Obsidian等）
- 📄 转换为PDF或Word文档
- 🔗 分享给他人

### 技巧3: 选择最佳时机
- 🌐 **原网页内容**: 页面完全加载后导出
- 🤖 **AI提炼**: 页面内容稳定时导出效果最佳

## 🆘 常见问题

### Q1: 为什么AI提炼模式报错？
**A**: 检查是否配置了API Key：
1. 打开扩展设置
2. 选择AI模型
3. 输入对应的API Key
4. 确保Key有效

### Q2: 导出的文件在哪里？
**A**: 通常在 **下载** 文件夹中。具体位置取决于浏览器设置。

### Q3: 可以自定义导出格式吗？
**A**: 当前支持Markdown格式。导出后可用其他工具转换为PDF或其他格式。

### Q4: 导出大型网页会很慢吗？
**A**: 
- 原网页内容: 很快（取决于网页大小）
- AI提炼: 需要调用AI模型，耗时较长

### Q5: 能否批量导出多个页面？
**A**: 目前不支持。需要逐页操作。
